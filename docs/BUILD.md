# Building

## Requirements

- CMake 3.25+
- Clang (C++20)
- Ninja
- [just](https://github.com/casey/just)

## Compilation

```bash
# Configure
just configure dev         # Debug -> build/
just configure release     # Release -> build-release/
just configure sanitize    # RelWithDebInfo + ASan/UBSan -> build-sanitize/

# Build
just build dev
```

Outputs in `build/bin/`:
- `arx-pistor` — CLI binary
- `arx_pistoris` (`.dll`/`.so`) — C API shared library produced by the `arx_pistoris_c` target

Outputs in `build/lib/`:
- `arx_pistoris_cpp` (`.lib`/`.a`) — C++ static library
- `arx_pistoris` import/static artifact as generated by the platform toolchain

## Installing The CLI

End-user installation is CLI-only. The installed command is `arx-pistor`; library headers and
library targets are build-tree integration details and are not installed.

From source:

```bash
cmake --preset release
cmake --build --preset release
cmake --install build-release --prefix <install-prefix>
```

The executable is installed to `<install-prefix>/bin`. Put that directory on `PATH` to run
`arx-pistor` from any terminal.

From GitHub Releases, use the install script for your platform:

```powershell
irm https://raw.githubusercontent.com/Merxtef/arx-pistoris/main/scripts/install.ps1 | iex
```

```bash
curl -fsSL https://raw.githubusercontent.com/Merxtef/arx-pistoris/main/scripts/install.sh | sh
```

Defaults:

- Windows: installs to `%LOCALAPPDATA%\arx-pistoris\bin` and adds that directory to the user `PATH`.
- Linux: installs to `$HOME/.local/bin`. If needed, the script appends that directory to `~/.profile`
  or prints the manual `PATH` command.

Open a new terminal after installation if the script changed `PATH`.

Release archives are expected to be named:

- `arx-pistoris-windows-x64.zip`
- `arx-pistoris-linux-x64.tar.gz`

## Linking

### Shared Library (C API)
Link against the CMake target `arx_pistoris_c` and include headers from `include/`. The produced shared library is named `arx_pistoris`.

### Static Library (C++ API)
Link against the CMake target `arx_pistoris_cpp` and include headers from `include/`. The target defines `ARX_PISTORIS_CPP_API`, which is required by `arx_pistoris/pistoris.hpp`.
